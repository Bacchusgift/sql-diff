## 如何使用

### 本地预览

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 访问 http://localhost:5173
```

### 构建部署

```bash
# 构建生产版本
npm run docs:build

# 预览生产版本
npm run docs:preview
```

### 部署到生产环境

可选择以下平台:
- GitHub Pages
- Vercel
- Netlify
- Cloudflare Pages

详见 `DOCS.md` 中的部署说明。
